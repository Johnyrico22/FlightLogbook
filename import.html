Import Flight Data

Please format your CSV file with one row per flight entry and the following headers:

  • date (e.g., YYYY-MM-DD, MM/DD/YYYY, etc.)
  • aircraft
  • registration
  • name
  • designation
  • departurePoint
  • departureTime (e.g., HH:mm)
  • arrivalPoint
  • arrivalTime (e.g., HH:mm)
  • flightType (either "single" or "dual")
  • instrument (HH:mm)
  • takeOffs (integer)
  • landings (integer)
  • dayHours (HH:mm)
  • nightHours (HH:mm)
  • tacoFinish (number with one decimal)
  • hobbsFinish (number with one decimal)
